## Project Vision
"Stop Scrolling, Start Pulsing."
CyberPulse AI is a fully automated newsletter system designed to cure AI FOMO for US-based users. It curates high-signal AI news daily and delivers it via Beehiiv, with traffic driven by an automated Twitter presence.

## Tech Stack
- **Curation Engine**: OpenClaw + Reddit/HackerNews/X API Scrapers.
- **Editorial Brain**: Claude 3.5 Sonnet / Opus for synthesis and "Cyber" tone.
- **Distribution Platform**: [Beehiiv](https://www.beehiiv.com/) via API.
- **Traffic Driver**: X (Twitter) Automation Engine (Managed by Lobster II).

## Newsletter Architecture (Cyberpunk Theme)
1. **The Neural Summary (TL;DR)**: 5 bullet points of today's biggest AI shifts.
2. **The Triple Deep-Dive**: 3 hand-picked long-reads with expert commentary.
3. **Lobster's Power-Up**: Featured OpenClaw skill or AI tool of the day.
4. **Cyber Ramble**: Informal industry gossip and hot takes.
